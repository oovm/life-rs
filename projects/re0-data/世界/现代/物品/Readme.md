## 初始天赋

增加初始属性的天赋

带有 `初始` 字段

```re0
/// 初始可用属性点+8
天赋 天命 {
    初始: {
        属性: 8
    }
}
```

## 一次性天赋

生效一次并移除

带有 `效果` 字段

```re0
/// 快乐+2
天赋 皆大欢喜 {
    效果: {
        快乐: 2
    }
}
```

## 条件触发天赋

满足触发条件后生效一次并移除

带有 `触发` 字段

```re0
/// 18岁时魅力+2
天赋 十八变 {
    触发: 年龄 == 18
    效果: {
        魅力: 2
    }
}
```

## 常驻天赋

带有 `常驻` 字段

- 常驻也可以有条件触发, 但是触发后会被移除
    - 如果不需要移除应该使用 `条件` 字段
- 常驻不能加随机属性

```
/// 快乐>10时其他属性+2
天赋 乐不思蜀 {
    条件: 快乐 > 10
    效果: {
        财富: 2,
        魅力: 2,
        体质: -2,
        智力: -2
    }
}
```
